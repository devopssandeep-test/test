process to add WebUI string :

				1)Add String in WebUICOnstant.java with serial ID at the bottom and before comment .

				2)Add same string in GlobalDictionary_WebUI.xls.

				3) Add same string in 8 properties files with key and value combination value must be in its language format
				
				4)	Generate xls sheet using i18builder tool (how to use i18 builder you can find in its readme.text file )
